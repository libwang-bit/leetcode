## 最大乘积子数组
(https://leetcode-cn.com/problems/maximum-product-subarray/submissions/)
```
描述:
给你一个整数数组 nums ，请你找出数组中乘积最大的连续子数组（该子数组中至少包含一个数字），并返回该子数组所对应的乘积。

 

示例 1:

输入: [2,3,-2,4]
输出: 6
解释: 子数组 [2,3] 有最大乘积 6。
示例 2:

输入: [-2,0,-1]
输出: 0
解释: 结果不能为 2, 因为 [-2,-1] 不是子数组。

```

Java 代码:

```
class Solution {
    public int maxProduct(int[] nums) {
        int n = nums.length;
        int max[] = new int[n];
        int min[] = new int[n];

        max[0]= nums[0];
        min[0] = nums[0];
        int result = Integer.MIN_VALUE;
        for (int i = 1; i < nums.length; i++) {
            max[i] = Math.max(Math.max(min[i - 1] * nums[i], nums[i] * max[i - 1]), nums[i]);
            result = Math.max(max[i], result);
            min[i] = Math.min(max[i - 1] * nums[i], Math.min(nums[i], nums[i] * min[i - 1]));
        }
        return Math.max(max[0], result);
    }
}
```
