## 二叉树的垂直遍历
(https://leetcode-cn.com/problems/binary-tree-vertical-order-traversal)

描述:
```
给定一个二叉树，返回其结点 垂直方向（从上到下，逐列）遍历的值。

如果两个结点在同一行和列，那么顺序则为 从左到右。

示例 1：

输入: [3,9,20,null,null,15,7]

   3
  /\
 /  \
9   20
    /\
   /  \
  15   7 

输出:

[
  [9],
  [3,15],
  [20],
  [7]
]

```

解法:

```
首先层序遍历, 用一个offsetMap 记录每个节点的偏移量.根节点为0。
左节点的偏移 = 根节点的偏移 - 1；
右节点的偏移 = 根节点的偏移 + 1；
然后用一个treemap，key是偏移量, value 是一个数组，按照层序遍历 往后追加的.
这里利用了treemap 默认是按照key 的升序排列,所以我们最后直接遍历map即可。
```

Java 代码:
```
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public List<List<Integer>> verticalOrder(TreeNode root) {
        if (root == null) {
            return new ArrayList();
        }
        TreeMap<Integer, List<Integer>> map = new TreeMap();
        Map<Integer, Integer> offsetMap = new HashMap();
        Queue<TreeNode> q = new LinkedList();
        q.offer(root);
        offsetMap.put(root.val, 0);
        List<Integer> l = new ArrayList();
        l.add(root.val);
        map.put(0, l);
        while(q.size() != 0) {
            int size = q.size();
            for (int i = 0; i < size; i++) {
                TreeNode t = q.poll();
                int offset = offsetMap.get(t.val);
                if (t.left != null) {
                    q.offer(t.left);
                    offsetMap.put(t.left.val, offset - 1);
                    List tt = map.getOrDefault(offset - 1, new ArrayList());
                    tt.add(t.left.val);
                    map.put((offset - 1), tt);
                }
                if (t.right != null) {
                    q.offer(t.right);
                    offsetMap.put(t.right.val, offset + 1);
                    List tt = map.getOrDefault(offset + 1, new ArrayList());
                    tt.add(t.right.val);
                    map.put((offset + 1), tt);
                }
            }
        }

        List<List<Integer>> result = new ArrayList();
        
        for(Map.Entry<Integer, List<Integer>> entry : map.entrySet()) {
            result.add(entry.getValue());
        }
        return result;
    }
}
```
